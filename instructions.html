ExploreEase - India-Centric Travel Website

Adding Sample Packages

To add the sample Indian tour packages to your database, follow these steps:

  1. Make sure your PHP server is running
  2. Make sure your MySQL server is running
  3. Access the remove_test_package.php file through your web browser to remove any test packages and add all the new Indian tour packages
  4. You should see a confirmation message that packages were updated successfully
  5. Alternatively, you can access add_packages.php and add_more_packages.php individually to add packages
Note: If you run this script multiple times, you will have duplicate packages in your database.

Viewing the Website

After adding the packages, you can view the website:

  1. Access index.html to view the homepage
  2. Click on "Tour Packages" to see all available packages
  3. Click on "View Details" to see package details
  4. You can also view a preview of the new packages that will be added

Troubleshooting

If you encounter any issues:

  • Make sure your PHP and MySQL servers are running
  • Check that the database connection details in api/config/database.php are correct
  • If viewing directly from file system (without a server), API calls will not work
